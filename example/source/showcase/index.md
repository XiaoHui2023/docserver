# 功能演示

本组页面用于在浏览器中逐项核对 **MkDocs Material** 已启用的能力。

## 子页面

| 页面 | 内容 |
| --- | --- |
| [Markdown 排版](markdown.md) | 提示框、折叠、标签页、表格、代码高亮 |
| [导航与目录](navigation.md) | 左栏层次、右侧 TOC、返回顶部 |
| [搜索](search.md) | 全文搜索与关键词建议 |
| [静态资源](static-files.md) | 图片与同目录非 Markdown 文件 |

## 主题与配色

**配色风格**：宽屏在顶栏（搜索框左侧）下拉；窄屏在左侧导航最上方。默认 **GitHub 灰栏**，另含 A–L 共 12 套（见 **`THEME-PALETTES.md`**），偏好存浏览器 `localStorage`。太阳图标仍切换浅色 / 深色。
