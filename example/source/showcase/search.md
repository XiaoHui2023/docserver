# 搜索演示

## 如何使用

1. 点击顶栏 **搜索**（或按键盘 `/` 聚焦搜索框，视主题版本而定）。
2. 输入关键词，在下拉建议中选择页面。
3. 回车查看完整结果列表。
4. 顶栏搜索右侧 **分享** 按钮可复制当前页链接（中文路径保持可读，非乱码 `%` 编码）。

## 建议试搜的关键词

复制下表任一词到搜索框，应能命中对应页面：

| 关键词 | 预期命中 |
| --- | --- |
| `搜索演示词` | 本页 |
| `子路径` | [部署与子路径](../guides/advanced/config.md) |
| `标签页` | [Markdown 排版](markdown.md) |
| `术语表` | [术语表](../reference/glossary.md) |
| `docserver` | 多页（首页、安装等） |

## 唯一测试串

若需确认索引已更新，可搜索下面这一整串（仅在本页出现）：

```text
docserver-搜索-demo-7f3a
```

修改源文件后，在 `--watch` 模式下等待自动重建，再搜一次应仍有效。

## 说明

- 顶栏搜索：检索 **全文**（标题与正文），适合找具体内容。
- 左侧 **筛选导航**：只在目录树里缩小范围，适合文档很多、记得页面名或路径片段时快速定位。
- 搜索索引在 **构建时** 生成，纯静态部署同样可用，无需服务端动态接口。
- 中文标题与正文均可检索。
