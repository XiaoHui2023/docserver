# 导航与目录

## 左侧导航栏

- 结构来自源目录树：每个子文件夹对应侧栏一组；可折叠 / 展开查看子页（`navigation.expand`）。
- 文件夹的 **首页** 由该目录下的 `index.md`（或 `README.md` 同步而来）充当。
- 本示例中可见四层：**首页 → 指南 / 功能演示 / 参考 → 子页**。
- 侧栏分组标题优先取该组 `index.md` 的第一个 `#` 标题。

试试：折叠 / 展开「功能演示」分组，点击各子页切换（**instant** 无整页刷新，仅中间正文更新，顶栏与左侧目录保持不动；明暗模式在换页后应保持不变；悬停链接触发预取）。若仍整页白屏或配色闪回浅色，请关掉 example 重开并 Ctrl+F5。

## 正文路径条

- 页面标题上方显示层级路径（`navigation.path`），用于逐级回退。
- 某层目录 **没有** `index.md` / `README.md` 入口时，该层在路径条中为灰色且不可点（无悬停反馈）。

## 右侧本页目录

- 自动根据当前页标题层级生成（`#` / `##` / `###`）。
- 点击可跳到对应小节；长文见 [Markdown 排版](markdown.md)。
- 滚动时当前小节会在目录中高亮（`toc.follow`）。

## 顶栏

| 元素 | 作用 |
| --- | --- |
| 站点名 | 仅在顶栏显示，点击回到首页（侧栏不再重复） |
| 搜索框 | 全文检索（见 [搜索](search.md)） |
| 调色板图标 | 浅色 / 深色模式切换 |
| 回到顶部 | 长页滚动后出现 |

## 多层「小仓库」结构示意

```text
example/source/          ← 总源根（本示例）
├── README.md            ← 站点首页
├── guides/              ← 像独立文档包
│   ├── install.md
│   └── advanced/
│       └── config.md
├── showcase/            ← 功能演示包
│   └── …
└── reference/           ← 参考包
    └── glossary.md
```

真实项目中，每个 `guides/`、`reference/` 都可来自不同 Git 仓库，同步到同一源根后一次构建。
