# 术语表

| 术语 | 含义 |
| --- | --- |
| **源目录** | `-s` 指定的 Markdown 与静态资源根 |
| **输出目录** | `-o` 指定的构建结果，可直接部署 |
| **base-url** | 站点挂在子路径时的 URL 前缀 |
| **构建缓存** | 默认当前目录下 `.docserver-cache/`（可用 `--cache-dir` 指定），含中间 `docs/` 与 `mkdocs.yml` |
| **入口页** | 每层目录的 readme / index 类 Markdown |
| **Material** | MkDocs 使用的主题，提供搜索与双配色 |

搜索时可试：`术语表`、`base-url`、`工作区`。
